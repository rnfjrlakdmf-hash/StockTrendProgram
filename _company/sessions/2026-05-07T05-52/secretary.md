📱 영숙: 사장님, API 키 설정 및 필수 패키지 설치 안내 드립니다. 사용자분이 API 키 미설정, python3·필수 패키지 미설치로 어려움을 겪고 계시는군요. 걱정 마세요, 제가 상세하게 안내해 드리겠습니다! 😊

**1. API 키 설정 및 필수 패키지 설치**

*   **API 키 확인:** 먼저, LM Studio에서 API 키가 제대로 설정되었는지 확인해야 합니다. LM Studio를 실행하고, 설정 메뉴에서 API 키를 입력했는지 확인해 보세요. (API 키는 보안에 매우 중요하므로, 함부로 공개하지 마세요!)
*   **필수 패키지 설치:** 다음 명령어를 사용하여 필요한 패키지를 설치합니다. (python3 설치는 이미 되어있는지 먼저 확인해주세요.)

    ```bash
    pip3 install -r requirements.txt
    ```

    *   `requirements.txt` 파일은 프로젝트에 필요한 모든 패키지 목록을 담고 있습니다. 이 파일을 사용하여 필요한 패키지를 한 번에 설치할 수 있습니다.

*   **관련 문서 링크:**
    *   [LM Studio 공식 홈페이지](https://lmstudio.ai/)
    *   [requirements.txt 파일](sessions/2026-05-07T05-02/requirements.txt) (실제 파일 내용 확인)

**2. LM Studio 설치 및 설정**

*   **LM Studio 다운로드:** [LM Studio 공식 홈페이지](https://lmstudio.ai/) 에서 운영체제에 맞는 LM Studio를 다운로드합니다.
*   **LM Studio 설치:** 다운로드한 설치 파일을 실행하고, 화면의 지시에 따라 LM Studio를 설치합니다.
*   **LM Studio 설정:** LM Studio를 실행하고, API 키를 설정합니다. (LM Studio 설정 방법은 LM Studio 공식 홈페이지에서 자세한 내용을 확인할 수 있습니다.)

**3. 텔레그램 봇 연동 방법**

*   **텔레그램 봇 생성:** 텔레그램에서 @BotFather 봇을 검색하여 봇을 생성합니다.
*   **봇 토큰 획득:** 봇을 생성한 후, 봇 토큰을 획득합니다. (봇 토큰은 텔레그램 봇을 제어하는 데 사용되는 비밀 키입니다. 매우 중요하므로, 안전하게 보관하세요!)
*   **텔레그램 봇 토큰 설정:** LM Studio에서 텔레그램 봇 토큰을 설정합니다. (텔레그램 봇 토큰 설정 방법은 LM Studio 공식 홈페이지에서 자세한 내용을 확인할 수 있습니다.)

**4. 문제 해결**

*   **API 키 오류:** API 키가 올바르게 설정되었는지 다시 한번 확인해 보세요.
*   **패키지 설치 오류:** 패키지 설치 오류가 발생하면, 다음 명령어를 사용하여 pip를 업데이트해 보세요.

    ```bash
    pip3 install --upgrade pip
    ```

    그런 다음, 다시 패키지를 설치해 보세요.
*   **텔레그램 봇 연동 오류:** 텔레그램 봇 토큰이 올바르게 설정되었는지 다시 한번 확인해 보세요.
