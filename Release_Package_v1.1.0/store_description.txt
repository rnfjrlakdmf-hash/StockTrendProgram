[자세한 설명 (4000자 이내)]
"이 종목의 재무 데이터와 수급 현황은 어떨까?"
AI 비서가 24시간 실시간 데이터를 분석하여 리포트를 제공합니다.

🚀 주요 기능
1. 🤖 AI 실시간 데이터 질의응답
   - "삼성전자 수급 현황 알려줘", "반도체 관련 뉴스 요약해줘"
   - 뉴스, 재무제표, 시장 데이터를 종합해 AI가 객관적으로 답변해 드립니다.

2. 📊 주식 데이터 분석 리포트
   - 기업의 재무 상태, 수급(기관/외국인), 뉴스 심리를 분석하여 수치화된 리포트를 제공합니다.
   - 종목의 객관적 지표를 한눈에 파악하세요.

3. 📈 차트 패턴 분석 도움말
   - 헤드앤숄더, 이중바닥 등 주요 차트 패턴을 AI가 감지하여 정보를 제공합니다.
   - 과거 유사 패턴의 통계적 추이를 확인하세요.

4. 🚦 글로벌 시장 데이터 대시보드
   - 코스피, 나스닥 지수부터 환율, 금리, 유가까지!
   - 오늘의 시장 지표를 직관적인 UI로 한눈에 확인 가능합니다.

5. 💎 프리미엄 분석 리포트
   - 데이터 기반의 심층 분석 리포트를 제공합니다.
   - (구글 로그인 시 무료 조회 가능)

더 이상 복잡한 데이터 사이에서 방황하지 마세요. AI 주식 리서치 비서와 함께 객관적인 데이터 분석을 시작하세요!
