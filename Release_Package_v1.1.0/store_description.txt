🚀 주요 기능
1. 🤖 AI 실시간 투자 상담
   - "삼성전자 전망 어때?", "지금 뜨는 반도체 관련주 알려줘"
   - 뉴스, 재무제표, 시장 데이터를 종합해 AI가 알기 쉽게 답변해 드립니다.

2. 🏥 주식 건강검진 (Health Check)
   - 내가 가진 종목의 재무 상태, 수급(기관/외국인), 뉴스 심리를 점수로 진단합니다.
   - 위험한 테마주인지, 저평가 우량주인지 한눈에 파악하세요.

3. 📈 차트 패턴 자동 분석
   - 헤드앤숄더, 이중바닥 등 복잡한 차트 패턴을 AI가 자동으로 감지합니다.
   - 상승/하락 확률을 예측하고 매매 타이밍을 잡아드립니다.

4. 🚦 글로벌 시장 모니터
   - 코스피, 나스닥 지수부터 환율, 금리, 유가까지!
   - 오늘의 시장 분위기를 신호등(초록/노랑/빨간불)으로 직관적으로 보여드립니다.

5. 💎 프리미엄 예측 리포트
   - 지난주 AI의 예측이 얼마나 맞았는지 투명하게 공개합니다.
   - (구글 로그인 시 무료 열람 가능)

이제 전문가 리포트 찾아 헤매지 말고, AI 주식 상담사와 함께 똑똑한 투자를 시작하세요!
